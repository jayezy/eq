# EQMind - Emotional Intelligence Landing Page

A sleek, modern landing page for an emotional intelligence assessment and training platform. Built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Live Demo

**[View Live Site](https://landing-page-two-kappa-99.vercel.app)**

## Features

### Interactive EQ Assessment
- 10 multiple-choice questions to discover your emotional type
- Animated progress bar and smooth transitions
- 5 distinct emotional intelligence profiles:
  - **The Introspector** - Self-Aware
  - **The Connector** - Empathic
  - **The Anchor** - Regulated
  - **The Driver** - Motivated
  - **The Orchestrator** - Social
- Personalized results with strengths and growth opportunities

### Curated Book Recommendations
Top 10 emotional intelligence books including:
1. Emotional Intelligence - Daniel Goleman
2. Emotional Intelligence 2.0 - Travis Bradberry & Jean Greaves
3. Working with Emotional Intelligence - Daniel Goleman
4. The Language of Emotions - Karla McLaren
5. Permission to Feel - Marc Brackett
6. Primal Leadership - Goleman, Boyatzis & McKee
7. The EQ Edge - Steven J. Stein & Howard E. Book
8. Social Intelligence - Daniel Goleman
9. Nonviolent Communication - Marshall B. Rosenberg
10. Daring Greatly - Brene Brown

### Wisdom Section
Curated quotes from Daniel Goleman's groundbreaking book "Emotional Intelligence: Why It Can Matter More Than IQ"

### Modern Design
- Dark, minimal aesthetic with ambient gradient backgrounds
- Scroll-triggered animations using Framer Motion
- Fully responsive design
- Ultra-light typography with subtle transparency effects

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Deployment:** Vercel

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Project Structure

```
src/
  app/
    page.tsx      # Main landing page with all sections
    layout.tsx    # Root layout with metadata
    globals.css   # Global styles and Tailwind imports
```

## Sections

1. **Hero** - Gradient headline with call-to-action buttons
2. **EQ Assessment** - Interactive quiz with animated transitions
3. **Quotes** - Wisdom from Daniel Goleman
4. **Features** - Six core EI competencies
5. **Books** - Top 10 book recommendations
6. **CTA** - Email signup with gradient background
7. **Footer** - Navigation links
